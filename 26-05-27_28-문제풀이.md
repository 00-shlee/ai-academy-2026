## 문제 1
### 리스트 합계와 평균

``` python
# 다음 리스트는 어떤 학생의 5과목 시험 점수입니다.
# for문을 사용하여 총점과 평균을 출력하세요.
# (평균은 소수점 그대로 출력해도 OK)

scores = [85, 90, 75, 88, 92]

# 출력 예시:
# 총점: 430점
# 평균: 86.0점

total = 0
for score in scores:
    total += score

average = total / len(scores)
print(f'총점: {total}점')
print(f'평균: {average}점')
```

## 문제 4
### 입력 받기 + 조건
``` python
# 다음 딕셔너리는 도서 제목과 재고입니다.
# 사용자에게 책 제목을 입력받아:
# - 재고가 있으면(1권 이상) "○○ 책은 ○권 남았습니다."
# - 재고가 없으면(0권) "○○ 책은 품절입니다."
# - 등록되지 않은 책이면 "등록되지 않은 도서입니다."

books = {
    "어린왕자": 5,
    "데미안": 0,
    "1984": 3,
    "노인과 바다": 0,
    "이방인": 7
}

# 실행 예시:
# 책 제목을 입력하세요: 어린왕자
# 어린왕자 책은 5권 남았습니다.

# 책 제목을 입력하세요: 데미안
# 데미안 책은 품절입니다.

# 책 제목을 입력하세요: 햄릿
# 등록되지 않은 도서입니다.

book = input("책 제목을 입력하세요.: ")

if book in books:
    if books[book] >= 1:
        print(f'{book} 책은 {books[book]}권 남았습니다.')
    else:
        print(f'{book} 책은 품절입니다.')

else:
    print("등록되지 않은 도서입니다.")
```

## 문제 5
### 함수 + 리스트 안 딕셔너리

``` python
# 다음 리스트에는 직원 정보가 딕셔너리로 저장되어 있습니다.
# 1) 월급에 따라 등급을 반환하는 함수를 만드세요.
#    - 5000000 이상: "임원"
#    - 3000000 이상 5000000 미만: "팀장"
#    - 2000000 이상 3000000 미만: "사원"
#    - 2000000 미만: "인턴"
# 2) 모든 직원을 순회하며 "이름 - 부서: ○○, 월급: ○○원, 등급: ○○" 형식으로 출력하세요.

employees = [
    {"name": "김철수", "dept": "개발", "salary": 5500000},
    {"name": "이영희", "dept": "마케팅", "salary": 3200000},
    {"name": "박민수", "dept": "개발", "salary": 2300000},
    {"name": "최지영", "dept": "디자인", "salary": 1800000},
    {"name": "정수현", "dept": "개발", "salary": 4100000}
]

# 출력 예시:
# 김철수 - 부서: 개발, 월급: 5500000원, 등급: 임원
# 이영희 - 부서: 마케팅, 월급: 3200000원, 등급: 팀장
# 박민수 - 부서: 개발, 월급: 2300000원, 등급: 사원
# 최지영 - 부서: 디자인, 월급: 1800000원, 등급: 인턴
# 정수현 - 부서: 개발, 월급: 4100000원, 등급: 팀장


def grade(salary):
    if salary >= 5000000:
        return "임원"
    elif salary >= 3000000:
        return "팀장"
    elif salary >= 2000000:
        return "사원"
    else:
        return "인턴"

for item in employees:
    name = item['name']
    dept = item['dept']
    salary = item['salary']
    member_grade = grade(salary)
    print(f'{name} - 부서: {dept}, 월급: {salary}원, 등급: {member_grade}')
```
## 문제 6
### 함수 + 분석
``` python
# 다음 리스트에는 학생들의 과목별 점수가 딕셔너리로 저장되어 있습니다.
# 각 학생의 평균 점수를 계산하고, 평균 점수에 따른 등급을 반환하는 함수를 만들어 사용하세요.
# 등급 기준: 90 이상 A, 80 이상 B, 70 이상 C, 60 이상 D, 그 미만 F

students = [
    {"name": "강예진", "scores": [95, 88, 92, 87]},
    {"name": "선영표", "scores": [78, 85, 80, 92]},
    {"name": "김민수", "scores": [70, 65, 72, 68]},
    {"name": "조은유", "scores": [60, 55, 70, 65]}
]

# 출력 예시:
# 강예진 - 평균: 90.5점, 등급: A
# 선영표 - 평균: 83.75점, 등급: B
# 김민수 - 평균: 68.75점, 등급: D
# 조은유 - 평균: 62.5점, 등급: D

def grade(average):
    if average >= 90:
        return "A"
    elif average >= 80:
        return "B"
    elif average >= 70:
        return "C"
    elif average >= 60:
        return "D"
    else :
        return "F"

# 평균 점수를 소수점 첫째 자리까지 작성하고 싶으면 {average:.1f}로 작성
for student in students:
    name = student["name"]
    scores = student["scores"]
    average = sum(scores) / len(scores)
    return_grade = grade(average)
    print(f'{name} - 평균:{average}점, 등급:{return_grade}')

```
