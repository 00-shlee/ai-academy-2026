### 1) for문 기본 구조

for문은 정해진 횟수만큼 반복할 때 자주 사용함

기본 구조:

```text

```

### 2) range() 함수

range()는 반복 횟수를 생성하는 함수

| 코드 | 의미 |
|---|---|
| range(5) #range(끝) | 0~4; 0부터 끝-1까지 |
| range(1, 5) #range(시작, 끝) | 1~4 |
| range(1, 10, 2) #range(시작, 끝, 간격) | 1부터 9까지 2만큼 간격을 가짐 |

==> 시작부터 끝-1까지 간격만큼 점프

ex)

```python
for i in range(10, 0, -1):
 print(i) #10부터 1(0의 직전)까지 -1의 간격(거꾸로) 출력됨
```

ex 3.) 리스트 안의 숫자 중 60 미만인 숫자를 60으로 수정

```python
scores = [50, 80, 40, 90] #[60, 80, 60, 90]으로 출력

for i in range(len(scores)):
 if scores[i] < 60:
  scores[i] = 60

print(scores)
```

'''
해당 코드를 이해하려면
scores = [50, 80, 40, 90]이라는 리스트가 있는데
len으로 해당 리스트의 개수를 구한 뒤, # ==> len(4)
range로 0부터 시작해서 4 직전까지의 숫자를 만들고,
==> len(scores) = 4이기 때문 / scores의 요소의 개수가 총 4개니까 range(len(scores)) = range(4)가 됨
for문으로 그 숫자를 i라는 변수에 하나씩 넣어 반복하면서
i를 인덱스로 사용해 scores[i]값이 60미만이면 60으로 변환한 뒤,
==> scores[i] = 60은 원본 리스트의 i번째(if 조건에 해당하는 것)를 60으로 덮어 씀
반복이 끝나면 수정된 리스트를 한 번 출력하는 것
'''

```python
for score in scores:
 if score < 60:
  print(scores.index(score))
  scores[scores.index(score)] = 60
print(scores)
```

'''
해당 코드는 위의 것보다 조금 더 복잡한데
for문으로 scores에서 score의 값을 하나씩 꺼내 반복하는데,
if 문에서 해당 score의 값이 60미만이라면
.index(score)에서 해당 score이 위치하는 index 번호를 찾고 print로 출력한 뒤에
그 인덱스 번호에 해당하는 score 값을 60으로 변경해서
반복이 끝난 뒤, 수정된 리스트를 한 번에 출력하는 것

단점이 있는데,
.index()는 중복되는 값이 있으면 가장 먼저 나온 인덱스 위치만 반환하기 때문에
중복되는 값이 있어도 첫 번째 위치만 알려 주므로 나머지 중복 위치는 찾지 못함
'''

### 7) break와 continue

- break: 반복문을 즉시 종료
- continue: 현재 반복만 건너뛰고 다음 반복으로 이동

ex 1.)

```python
for i in range(10):
 if i == 5:
  break
  print(i) # 값 없음 for문이 break에서 탈출해서 print(i)의 출력값을 구할 수 없음
```

수정하면

```python
for i in range(10):
 print(i)
 if i == 5:
  break # for문에서 반복되는 i값이 계속 누적되어 i가 5가 될 때까지 출력된 뒤, break로 for문 종료됨 # 0 1 2 3 4 5
```

ex 2.)

```python
count = 1
while count <= 5:
 print(count)
 count += 1 # 1 2 3 4 5
```

=> count값이 1부터 시작해서, while 조건 count <= 5가 될 때까지 반복,
print(count) 먼저, count += 1 나중, 입력되는 count가 5일 때 출력 후 6으로 증가, 다음 회차에서 조건이 거짓이 되어 종료

ex 3.)

```python
count = 1
while True:
 print(count)
 count += 1
 if count == 5:
  break #1 2 3 4
```

(무한반복)

=> count값이 1부터 while문을 통해 무한히 반복되는데, while문 안에 있는 if문으로 인해 count값이 5가 되면 해당 while문이 멈춤

=> 보충 설명: print(4) 실행했을 때 4가 출력이 됐고, count += 1이 실행되어 5가 됐는데 이때 if count == 5 문을 충족해 break를 통해 멈춤
따라서 실행이 완료된 4까지만 출력값에 포함되고, 5는 포함되지 않음
(print(5)가 되기 전 if문을 충족해 while문이 종료되었기 때문)

ex 4.)

```python
count = 0 
while count <= 4:
 count += 1
 if count == 3:
  continue
 print(count) # 1 2 4 5
```

=> count가 0부터 시작이지만, 매 회차마다 print보다 count += 1이 먼저 실행되어 1부터 출력됨. count가 3이 되는 회차에서는 continue로 해당 회차 출력되지 않음  5까지 출력되는 이유는 while 문 조건 검사 시점에 count = 4이고, 5로 증가해서 출력이 되기 때문, 다음 회차에는 count = 5가 되어 거짓이 되었기 때문에 종료
(출력 count 값은 실행값의 +1; count += 1을 통해 알 수 있음)

ex 5.)

```python
for i in range(1,6):
 if i == 3:
  continue
 print(i) # 1 2 4 5
```

1부터 6의 직전까지 i를 반복하는데, i가 3일 때는 제외를 하고 값을 출력해야 하니까 1 2 4 5가 됨

## 2. 딕셔너리(Dictionary)
### 키(key)와 값(value)을 한 쌍으로 저장하는 자료형

ex) 
```python
student = {
    "name": "홍길동",
    "age": 20,
    "major": "AI"
}

print(student)
```
- 각각 "name", "age", "major"가 key, "홍길동", 20, "AI"가 value에 해당
- list처럼 index로 데이터를 찾는 게 아닌, key를 이용해 value 값을 찾음
- 중괄호 {} 사용

### 1) 데이터 가져오기
key를 이용해 value의 값을 가져올 수 있음
```python
student = {
    "name": "홍길동",
    "age": 20
}

print(student['name']) # 홍길동
print(student['age']) # 20
```

### 2) 데이터 추가 및 수정
딕셔너리는 새로운 데이터를 추가하거나 기존 데이터를 수정할 수 있음
``` python
student = {
    "name": "홍길동",
    "age": 20
}

student["age"] = 21
student["major"] = "컴퓨터공학"

print(student) 
#{'name': '홍길동', 'age': 21, 'major': '컴퓨터공학'}
```

### 3) 데이터 삭제
``` python
student = {
    "name": "홍길동",
    "age": 20,
    "major": "AI"
}

del student['major'] 

print(student) #{'name': '홍길동', 'age': 20}
```
remove는 list 전용 메서드이기 때문에 del을 사용함

### 4) 딕셔너리 반복문
``` python
student = {
    "name": "홍길동",
    "age": 20,
    "major": "AI"
}

# key 가져오기
for key in student:
  print(key)
  # print(student[key]) key에 해당하는 value를 가져옴
 
# value 가져오기
for value in student.values():
  print(value)

# key, value 가져오기
for item in student.items():
  print(item)

#.values()와 .items()는 각각 dic의 method
```

## 3. 함수(Function)
### 1) 함수란? 
특정 작업을 여러 번 사용할 수 있도록 묶어놓은 코드

ex) "안녕하세요"를 출력하는 코드를 여러 번 작성하는 대신, 함수로 만들어두면 필요할 때마다 쉽게 사용할 수 있음

### 2) 기본 구조
``` python

    실행할 코드
```

``` python

```
를 만들 수 있음

### 3) 매개 변수
함수는 데이터를 전달 받아서 사용할 수 있는데,

이때 함수 안으로 전달되는 값을 저장하는 변수를 매개변수라 칭함


## 예외 처리

``` python

user_input = input("숫자를 입력해 주세요.")

# isdigit.()은 문자열의 method
# isdigit이란 "문자열".isdigit() 형태로 작성되는데, 해당 "문자열이" 숫자; 양의 정수로만 이루어져 있는지 T/F로 답함

# 해당 코드가 작동하는 이유는 user_input이 input값을 isdigit() 전에 int로 형변환 되지 않은 문자열의 형태라 해당 method의 조건 충족

# result에서는 문자열과 10이라는 숫자를 더하는 게 불가능하기 때문에 바로 위에서 number라는 변수를 만들어 user_input을 int로 형변환
if user_input.isdigit():
    number = int(user_input)
    result = number + 10
    print(f'결과: {result}')
else:
    print("숫자만 입력해 주세요.")
```
