# Python 숫자 자료형

## 2진수, 8진수, 16진수 (모두 int로 표기; class 'int')

### 2진수 (Binary)

- 사용 숫자: 2개 (0, 1)
- 각 자리값은 2의 거듭제곱 (오른쪽부터)
- ex) 1011 = 2<sup>3</sup>(1개) + 2<sup>2</sup>(0개) + 2<sup>1</sup>(1개) + 2<sup>0</sup>(1개) = 8 + 2 + 1 = 11

### 8진수 (Octal)

- 사용 숫자: 8개 (0~7)
- 각 자리값은 8의 거듭제곱
- ex) 1011 = 8<sup>3</sup>(1개) + 8<sup>2</sup>(0개) + 8<sup>1</sup>(1개) + 8<sup>0</sup>(1개) = 512 + 8 + 1 = 521
- ex) 345 = 8<sup>2</sup>(3개) + 8<sup>1</sup>(4개) + 8<sup>0</sup>(5개) = 3×64 + 4×8 + 5×1 = 229

### 16진수 (Hexadecimal)

- 사용 숫자: 16개 (0~9, A, B, C, D, E, F); 10~15를 한 자리 숫자로 표현하기에 마땅한 숫자 기호가 없어 알파벳을 빌림
- 각 자리값은 16의 거듭제곱
- ex) 2A = 16<sup>1</sup>(2개) + 16<sup>0</sup>(10개) = 2×16 + 10×1 = 42
- ex) FF = 16<sup>1</sup>(15개) + 16<sup>0</sup>(15개) = 15×16 + 15×1 = 255

### 진법 표기 방법

각 진법이 섞이면 헷갈리니까

- 2진수는 `0b1111` 혹은 `0B1111`
- 8진수는 `0o345` 혹은 `0O345`
- 16진수는 `0xFF` 혹은 `0XFF`로 표기

### 원리

d<sub>k</sub>d<sub>k-1</sub>…d<sub>1</sub>d<sub>0</sub>일 때, N<sup>0</sup>×d<sub>0</sub> + N<sup>1</sup>×d<sub>1</sub> + N<sup>k-1</sup>×d<sub>k-1</sub> + N<sup>k</sup>×d<sub>k</sub>

## 복소수 (complex number)

실수와 허수를 합친 수 (complex로 표기; class 'complex')

### 표기 예시

- a+bj, c-dj (실수부: a, c / 허수부: b, d)
  - 2+3j, 4j = 2+3i, 4i (각각 실수부는 2와 0, 허수부는 3과 4)

> ⚠️ 주의할 건 2+i라고 해서 `2+j`라고 적으면 `j`가 변수명이 되어 버려서, 꼭 j 앞에는 숫자가 (허수부의 숫자가) 붙어야 함
> ex) `2+1j`

### 주요 메서드

- z가 3+4j일 때, `print(abs(z))` = 5.0 / `abs()` = 원점에서의 거리
- `print(z.conjugate())` = 3-4j / `n.conjugate()` = n이라는 복소수의 켤레복소수를 구함
