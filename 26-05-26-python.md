논리 연산자

논리 연산자에는 and, or, not이 있음.
and와 or은 양옆에 항이 2개 오는 이항 연산자이고,
not은 하나의 항에 적용되는 단항 연산자(피연산자를 bool로 평가한 뒤, 반대 값 반환/ 결과는 항상 True 또는 False)

 - and는 양쪽 조건이 모두 True일 때만 True가 됨
```
True and True → True
True and False → False
False and True → False 
False and False → False
```

 - or은 둘 중 하나라도 True이면 True가 됨
```
True or True → True
True or False → True
False or True → True
False or False → False
```

 **단락 평가(short-circuit evaluation): python이 결과가 이미 정해진 경우, 나머지 안 봄
```
ex) False and (어떤 거든) -> 바로 False(오른쪽 안 봄) 
   True or (어떤 거든) -> 바로 True(오른쪽 안 봄)
```

 - not은 True를 False로, False를 True로 반대로 바꿈

```python
0        # False처럼 취급
0.0      # False처럼 취급
""       # 빈 문자열 → False처럼 취급
[]       # 빈 리스트 → False처럼 취급
None     # 값이 없음 → False처럼 취급
```
(0이 항상 False라는 것은 아니고, 조건문이나 논리 연산에서 False처럼 판단)

위 list처럼 요소의 숫자가 적으면 [0], [2]라고 적어도 무방하지만, 요소의 숫자가 몇 개인지 확실하지 않을 때 '마지막 요소'를 출력하라고 하면 '-1'이라고 입력하면 맨 뒤의 요소의 값이 도출됨

*슬라이싱은 26-05-18-python에 자세히 설명

```python
ex) a = 'abcdefg'
print(a[0:-1:2]) #ace / a의 0번째부터 -1 위치 직전까지, 2칸씩 이동하면서 각 요소를 가져옴, 맨 뒤의 요소 값은 포함되지 않음
```

 ㄴ 만약 a[::2]라면 처음부터 맨 뒤까지 2칸씩, a[::-1]은 문자열을 뒤집음(뒤에서부터 한 글자씩 가져오기 때문에 문자열이 뒤집힘)

```python
ex-1, 슬라이싱) a = 'hello', print(a[0:3]) #'hel' (0번째부터 3번째 직전까지)
ex-1.1) print(a[-1]) #'o'(마지막 글자), print(a[-3:]) #'llo'(뒤에서 3글자)
print(a[:-2]) #'hel'(뒤에서 2글자를 제외한 전체)
ex-2, 생략 가능) a = 'hello world', print(a[:5]) #'hello' (0번째부터 5번째 직전까지)
ex-2.1) print(a[6:]) #'world' (6번째부터 끝까지), print(a[:]) #'hello world' -> 전체 복사
```

3) list.pop(): 마지막 요소를 삭제하면서 해당 값을 반환함 / 
pop(index): 해당 index 위치에 있는 값을 꺼내서 삭제
