# RNN 이해하기

## RNN의 특징
- 이전의 데이터가 다음 데이터에 영향을 주는 Sequential한 데이터를 학습 시키기 위해 등장 (ex : 문장,음악,주식,날씨 등)
- 입력 데이터를 시간순에 따라서 쪼개고 순차적으로 집어 넣는 방식을 채택
- 현재 입력 데이터와 직전(과거) 메모리를 합산하는 수식으로 계산

### 학습
- 모델 설계(구조 정의)
- 학습 방법 설정
- 학습 및 평가

``` python
from tensorflow.keras.models import Sequential # 모델의 뼈대
from tensorflow.keras.layers import InputLayer # 입력층
from tensorflow.keras.layers import Dense # 뉴런의 묶음
from tensorflow.keras.layers import SimpleRNN # RNN 층
from tensorflow.keras.layers import Dropout # 과적합 방

# 뼈대 생성
model = Sequential()

# 입력층
model.add(InputLayer(shape=(10,1))) # 시퀀스의 길이, 입력 특성 수

# 중간층
model.add(SimpleRNN(units=64)) # 순환신경망 레이어 추가, 활성화 함수 생략 가능

# 출력층
model.add(Dense(units=1, activation='sigmoid')) # 이진 분류(긍정 부정)

# 모델 구조 확인
model.summary()

# 학습 방법 설정
model.compile(
    loss = 'binary_crossentropy', # 이진 분류(모델의 손실값을 파악하는 도구)
    optimizer = 'adam', # 손실값을 기반으로 방향을 알려 주는 도구
    metrics = ['accuracy'] # 정확도 평가 지표 추가
)

# 학습 및 평가
simple_his = model.fit(
    X_train_vec, # 학습용 문제
    train_df['label'], # 학습용 정답
    epochs=40,  # 학습 횟수
    batch_size=128, # 학습 시 한 번에 사용하는 샘플 수
    validation_split=0.2 # 검증 데이터 분할
)

# 학습 결과 시각화
import matplotlib.pyplot as plt

plt.figure(figsize = (10, 5)) # 가로 세로 사이즈
plt.plot(simple_his.history['accuracy'], label='train_acc') # 학습 손실값
plt.plot(simple_his.history['val_accuracy'], label='val_acc') # 검증 손실값
plt.legend() # 범례
plt.show()
```

### SimpleRNN과 현재 토크나이저의 문제점
- SimpleRNN은 시퀀스가 길어질수록 초반 단어에 대한 영향이 약화됨 => 각 시점에 따라 다른 영향력을 관리할 필요가 있음
- 현재 토크나이저는 단순 사전 숫자로 인코딩 된 숫자를 학습하기 때문에 단어가 가진 본질적 의미를 학습하지 못함 => 각 단어의 의미를 정교한 숫자로 변경할 필요가 있음
