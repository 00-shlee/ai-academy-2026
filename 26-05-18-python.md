> 문자열, 슬라이싱, 자주 쓰는 메서드, 그리고 클래스/객체/메서드 개념까지

## 📌 문자 자료형이란?

문자 자료형: 순서가 있는 시퀀스(sequence) 자료형

- **시퀀스(sequence)**: "순서가 있고, 번호(index)로 하나하나 꺼낼 수 있는 자료형"
- 시퀀스 자료형: `str`, `list`, `tuple`, `range`
- 비시퀀스: `dict`, `set`

### 시퀀스의 3가지 특징

1. 순서가 있음 (order)
2. 인덱스 접근 가능 (indexing)
3. 슬라이싱 가능 (slicing)

---

## ✂️ 슬라이싱 (Slicing)

문자열이나 리스트의 **일부분을 잘라내는 것**.

### 기본 문법

```python
변수[시작:끝]
```

- **시작**: 어디부터 자를지 → 이 위치 **포함 ✅**
- **끝**: 어디까지 자를지 → 이 위치 **포함 X ❌** (자르는 위치가 `끝-1`까지라고 생각)

### 예시

```python
a = "hello"
print(a[0:3])    # "hel"
print(a[1:4])    # "ell"
print(a[2:5])    # "llo"
```

### 생략 가능

```python
a = "hello world"
print(a[:5])     # "hello"     → 처음부터 4번까지
print(a[6:])     # "world"     → 6번부터 끝까지
print(a[:])      # "hello world" → 전체 복사
```

### 음수 인덱스 (뒤에서부터)

`a = "hello"`라면 뒤에서부터 -1로 시작해서 `h: -5`

```
 글자:   h    e    l    l    o
 양수:   0    1    2    3    4
 음수:  -5   -4   -3   -2   -1
```

```python
a = "hello"
print(a[-1])     # "o"     → 마지막 글자
print(a[-3:])    # "llo"   → 뒤에서 3글자
print(a[:-2])    # "hel"   → 뒤에서 2글자 빼고 전부
```

**자주 쓰는 패턴 — 확장자 꺼내기**

```python
파일명 = "image.png"
print(파일명[-3:])    # "png"
```

### 간격(step) 사용

```python
a[시작:끝:간격]
```

세 번째 자리가 **step(간격)**, 몇 칸씩 뛸지 결정.

- 간격 = 1 → 한 칸씩 (기본값, 다 가져옴)
- 간격 = 2 → 두 칸씩 (하나 건너뛰고 가져옴)
- 간격 = 3 → 세 칸씩 (둘 건너뛰고 가져옴)

```python
a = "abcdefgh"
print(a[0:8:2])    # "aceg"  → 2칸씩 건너뛰기
print(a[::2])      # "aceg"  → 처음부터 끝까지 2칸씩
print(a[::3])      # "adg"   → 3칸씩
```

### 🔄 문자열 뒤집기

```python
a = "hello"
print(a[::-1])    # "olleh"
```

> 위 특징들은 list, tuple 모두 똑같이 적용됨

## 📦 list vs tuple

> 근데 tuple은 수정이 안 된다고 했던 것 같은데, 슬라이싱이 가능해?
> → 부분적(읽기)으로 가능, 바꾸기는 불가능

### 읽기는 가능

```python
t = (10, 20, 30, 40, 50)
print(t[1:4])    # (20, 30, 40)  → 그냥 보기만 함, 원본 안 바꿈
```

### 수정은 불가능

```python
t = (10, 20, 30, 40, 50)
t[1:4] = (99, 99, 99)    # ❌ TypeError! tuple은 수정 불가
```

### 비교 정리

| 동작 | List | Tuple |
|------|:---:|:---:|
| 읽기 | ✅ | ✅ |
| 수정 | ✅ | ❌ |
| `append`, `remove` 등 | ✅ | ❌ |
| Index로 한 개 수정 (`a[0]=10`) | ✅ | ❌ |

### list 수정 예시

```python
li = [10, 20, 30, 40, 50]
li[1:4] = [100, 200, 300]
print(li)    # [10, 100, 200, 300, 50]
```

## 🛠 자주 사용하는 문자열 메서드

### 1) `lower()` / `upper()`
각각 소문자와 대문자로 해당 문자열 전체를 변경함.

```python
"Hello".upper()    # "HELLO"
"Hello".lower()    # "hello"
```

### 2) `find()` / `index()`
특징: `find()`는 찾고 있는 문자열이 없다면 "-1"로 반환, `index()`는 에러

> 에러는 최대한 발생시키지 않는 것이 좋음, 보다 안정적인 프로그래밍 환경 보장

### 3) `count()`
특정 문자열이나 숫자의 개수를 셀 때 사용.

```python
"hello".count("l")    # 2
```

### 4) `strip()`
데이터의 양쪽 끝의 지정된 문자들을 제거할 때 사용.

```python
"   hi   ".strip()    # "hi"
```

### 5) `replace()`
문자열의 특정 부분을 다른 문자로 교체.

```python
a = "hello, world"
print(a.replace('l', 'r'))    # "herro, worrd"
```

### 6) `split()`
문자열을 지정된 구분자를 기준으로 분할해서 list로 반환.

```python
'paullab,CEO,leehojun'.split(',')
# ['paullab', 'CEO', 'leehojun']
```

### 7) `join()`
문자열 list를 하나로 결합함.

```python
'!'.join(['paullab', 'CEO', 'leehojun'])
# 'paullab!CEO!leehojun'
```

### 8) `isalnum()`
해당 데이터 문자열의 모든 문자가 알파벳 또는 숫자로만 이루어져 있는지 확인.

### 9) `isdigit()`
해당 데이터 문자열의 모든 문자가 숫자인지 아닌지 확인.

### 10) `isalpha()`
해당 데이터의 문자열의 모든 문자가 알파벳인지 아닌지 확인.

### 11) `isascii()`
아스키코드인지 아닌지 확인.

### 12) 문자열 정렬 `rjust()` / `ljust()` / `center()`
각각 오른쪽, 왼쪽으로 밀어붙임, 가운데 정렬.

- `(n)`: "전체 길이가 n이 되도록" 공백을 채움, n보다 이미 길면 그대로

```python
"hi".rjust(10)     # "        hi"
"hi".ljust(10)     # "hi        "
"hi".center(10)    # "    hi    "
```

### 13) `zfill()`
데이터의 왼쪽 공간을 0으로 채워줌. 위와 같이 n의 길이는 데이터의 전체 길이.

- 문자의 길이가 `()`에 들어가는 숫자보다 이미 길 때, 0은 나타나지 않음.

```python
"42".zfill(5)      # "00042"
"12345".zfill(3)   # "12345"  (이미 더 김)
```

### 14) `translate()`
문자 단위에 변환(규칙) 테이블을 사용해서, 문자열을 변환함.

```python
규칙테이블 = str.maketrans('lr', '12')
'hello world'.translate(규칙테이블)
# 'he11o wo21d'
# → l을 1로, r을 2로 변환 시키라는 규칙을 만듦
```

### 15) `startswith` / `endswith`

```python
"hello".startswith("he")    # True
"hello".endswith("lo")      # True
"Hello".startswith("he")    # False  → 대소문자 구분!
```

- `startswith(x)` = 이 문자열이 x로 시작해?
- `endswith(x)` = 이 문자열이 x로 끝나?
- 결과는 항상 True/False (대소문자 구분해서 값이 나옴)

### 16) `capitalize()`

```python
"hello".capitalize()    # "Hello"  (첫 글자만 대문자)
```

### 17) `title()`

```python
"hello world".title()    # "Hello World"  (단어마다 첫 글자 대문자)
```

## 🏛 클래스 · 객체 · 메서드

> 갑자기 궁금해진 건데 클래스, 객체, method가 뭐야?

### 예시 코드

```python
class 학생:                          # ← 클래스 (설계도)
    def __init__(self, 이름, 점수):   # ← 메서드 (초기화 기능)
        self.이름 = 이름
        self.점수 = 점수

    def 합격여부(self):               # ← 메서드 (합격 판단 기능)
        if self.점수 >= 60:
            return f"{self.이름}: 합격"
        else:
            return f"{self.이름}: 불합격"

철수 = 학생("철수", 85)              # ← 철수 = 객체 1
영희 = 학생("영희", 45)              # ← 영희 = 객체 2

print(철수.합격여부())                # 객체.메서드() 호출
print(영희.합격여부())                # 객체.메서드() 호출
```

### 구조 매칭

| 구분 | 코드에서 |
|------|---------|
| **class** | `학생` |
| **객체** | `철수`, `영희` |
| **method** | `__init__`, `합격여부` |

### 🎯 비유로 정리

- **class**는 객체가 포함된 단체
- **객체**는 각 단체에 소속된 개인의 이름
- **method**는 단체가 제공하고, 개인이 (각자) 할 수 있는 활동의 이름

> 즉, **method = class 소속 객체가 할 수 있는 활동의 이름**
