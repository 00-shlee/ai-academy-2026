### 비정형 데이터 (Unstructured Data)

비정형 데이터는 미리 정의된 구조나 형식 없이 자유로운 형태로 존재하는 데이터를 말합니다. 정형 데이터(예: 관계형 데이터베이스의 테이블)와 달리, 데이터의 각 부분이 무엇을 의미하는지 파싱하기 어렵고, 전통적인 데이터베이스 시스템으로 관리하기 어렵습니다. 그러나 전체 데이터의 80% 이상을 차지하며, 분석을 통해 귀중한 통찰력을 얻을 수 있습니다.

#### 비정형 데이터의 주요 종류:

1.  **텍스트 데이터 (Text Data)**:
    *   **예시**: 이메일, 문서 (워드, PDF), 소셜 미디어 게시물, 웹 페이지, 뉴스 기사, 채팅 로그, 블로그 글, 고객 리뷰, 음성 기록.
    *   **특징**: 자연어로 구성되어 있어 의미를 이해하고 패턴을 추출하기 위해 자연어 처리(NLP) 기술이 필요합니다.

2.  **이미지 데이터 (Image Data)**:
    *   **예시**: 사진, 스캔된 문서, X-레이 이미지, 위성 이미지, 그림.
    *   **특징**: 픽셀 값으로 구성되며, 이미지 인식, 객체 탐지, 이미지 분류 등을 위해 컴퓨터 비전 기술이 사용됩니다.

3.  **오디오/음성 데이터 (Audio/Voice Data)**:
    *   **예시**: 음성 통화 녹음, 음악 파일, 팟캐스트, 음성 메모.
    *   **특징**: 시간 경과에 따른 소리 파형으로 표현되며, 음성 인식(STT), 화자 식별, 감정 분석 등에 활용됩니다.

4.  **비디오 데이터 (Video Data)**:
    *   **예시**: 영화, 감시 카메라 영상, 웹캠 녹화, 동영상 스트리밍.
    *   **특징**: 연속된 이미지(프레임)와 오디오 트랙으로 구성되며, 객체 추적, 행동 인식, 이벤트 감지 등에 컴퓨터 비전과 오디오 처리 기술이 복합적으로 사용됩니다.

5.  **소셜 미디어 데이터 (Social Media Data)**:
    *   **예시**: 트윗, 페이스북 게시물, 인스타그램 댓글, 유튜브 영상.
    *   **특징**: 텍스트, 이미지, 비디오 등 다양한 형태의 데이터가 혼합되어 있으며, 실시간으로 생성되고 방대합니다.

이러한 비정형 데이터는 빅데이터 시대의 핵심 자원으로, 머신러닝, 딥러닝, 자연어 처리 등의 고급 분석 기법을 통해 가치를 창출합니다.

### 말뭉치와 문서

### 토큰화
- 토큰화(Tokenize): 일정 단위로 문장을 쪼개는 작업
    1. 글자(char)
    2. 단어(띄어쓰기)
    3. 형태소
    4. N-gram

``` python
# 모든 문서를 토큰화 하자
tokens = [] # 20만 개의 리뷰의 모든 토큰을 담은 리스트
for doc in naver_df['리뷰'] : # 리뷰열을 꺼내서 반복
  for token in doc.split(" ") : # 띄어쓰기 중심으로 각 리뷰들을 토큰화
    tokens.append(token) # 잘라낸 개별 토큰을 리스트에 추가
```

### 특징 추출(수치화)
- 의미 있는 숫자를 뽑아내는 과정
- 빈도를 기반으로 숫자를 뽑아 보자
``` python
from collections import Counter

# 도구 생성
counter = Counter(tokens) #토큰화 된 데이터를 넣어서 생성

# 빈도 확인
counter.most_common(20) # 빈도가 높은 상위 20개 데이터 확인
```

### 배송이 포함된 데이터 클렌징

- 반복적인 글자(ㅋ,ㅎ,ㅠ,ㅇ 등)
- 실습 순서
    1. 패턴 만들기
    2. 문장 토큰화
    3. search 이용해서 패턴에 검출된 토큰은 제외하고 전체 리스트에 추가
