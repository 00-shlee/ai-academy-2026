# Python 입력과 출력

## input과 print

`a = input("a를 입력하세요.")` 를 풀어서 말하면 "a라는 변수에 값을 할당하는데, 그 값은 input이라는 함수의 결과/반환값임"

- 여기에서 "할당"에 대해 말하자면, `=` 기호를 사용해서 변수에 값을 넣어 주는 것을 "할당"이라고 하고 영어로 'assignment'라고 함 또 알아둬야 할 게 `a = 10`은 되는데 `10 = a`는 안 되는 이유가 오른쪽의 값을 왼쪽의 변수에 넣어야 해서, 즉, 왼쪽이 변수여야 함
- input 함수를 통해 입력 받은 모든 값은 문자열의 형태로 저장
  - ex) `a = input("a를 입력하세요.")`에서 input 함수에 a값을 10이라고 지정하면, a의 결과값이 '10'으로 나타남, 이때 `''`, `""`는 문자열(str)을 나타내는 기호
    - 이를 정수형으로 나타내려면 형 변환을 해야 함
    - ex) `b = int(input("a를 입력하세요."))`라고 했을 때, a에 20이라는 값을 할당했을 때, b를 출력하면 20이라는 결과값을 가짐, 이때 20은 기호가 없는 정수형(int)의 자료형으로 형태가 변환된 것을 알 수 있음

---

```python
age = input('age를 입력하세요.')
last_name = input('last_name을 입력하세요.')
first_name = input('first_name을 입력하세요.')
print('당신의 이름은', last_name + first_name, '입니다.')
print('당신의 나이는', age, '입니다.')
```

실행 결과:
```
age를 입력하세요.27
last_name을 입력하세요.이
first_name을 입력하세요.승현
당신의 이름은 이승현 입니다.
당신의 나이는 27 입니다.
```

위 코드에서 27이라는 나이와 이승현이라는 이름이 `''`에 감싸져서 나오지 않은 이유가 궁금했음

input으로 값을 할당했는데 왜 출력값에는 저렇게 자연스럽게 나오지? 했는데 따옴표는 코드 안에서 "이건 문자열임"라고 표시하는 기호일 뿐, 실제 값의 일부는 아님, 또한, print() 함수는 값의 내용 자체를 화면에 보여 주기 때문에 빠짐

```python
name = "이승현"
print(name)  # 이승현 - print는 내용만 보여 줌
name         # '이승현' - 변수만 입력하면 표현형으로 보여 줌
```

이처럼 따옴표는 코드에서만 보이고, print 함수로 출력할 때는 사라진다고 생각하면 됨

마지막 두 문장에 대해 말하자면 먼저, `+` 연산자가 보이는데 이는 두 문자열을 연결해서 하나의 문자열로 나타내는 연산자임(이때, 문자열끼리만 가능하고 숫자를 섞으면 에러 발생) 그리고 콤마(`,`)가 보이는데 콤마로 구분된 각 요소는 하나의 객체로 취급을 함, 따라서 문자열과 변수를 콤마로 구분해서 나열하면 자동으로 공백을 넣어 자연스럽게 연결해서 값을 출력(장점은 `+` 연산자와는 다르게 숫자와 문자열을 섞어도 에러가 나지 않음)

---

```python
age = 27
print('제 나이는', 27, '입니다.')
print('제 나이는', age, '입니다.')
```

를 넣으면
```
제 나이는 27 입니다.
제 나이는 27 입니다.
```
가 나오는데 여기에서 age라는 변수에 27이라는 값이 할당이 되었기 때문에 변수를 출력하든 값 자체를 출력하든 반환값이 똑같아짐

위처럼 출력함수인 print를 사용하는 방식에는 콤마로 연결하는 것 외에도 여러 개가 있음

- **f-string**: 문자열 중간에 변수를 끼워넣고 싶을 때, 중괄호 `{}`를 넣음으로 더 편하고 깔끔하게 만들 수 있음
  - `print(f'제 나이는 {age}입니다.')` 라고 하면 반환값은 동일하게 나옴
  - 위 문법을 사용함으로 간결하고 가독성이 좋아졌음 / 문장열 중간에 변수를 사용하는 것보다는 f-string 선호

- **format 이용**: `print('제 나이는 {}입니다.'.format(age))`
  - 중괄호로 문장 내 변수가 들어갈 자리를 표시하고, 뒤쪽에 format으로 문장 내부 각각의 중괄호에 들어갈 값을 순서대로 입력함

- **포맷코드**: `print('제 나이는 %d입니다.' % int(age))`
  - 잘 사용하는 코드는 아님, 알아만 두자
  - 각 포맷 코드의 이름 유래
    - `%c`: character(문자)
    - `%s`: string
    - `%d`: decimal(10진수 정수)
    - `%f`: float(실수)
